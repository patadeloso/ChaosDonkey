# Changelog

## [0.1.0] - 2026-04-02

Initial release of `shaunmcmanus/chaosdonkey`.

### Added
- Magento 2 CLI commands for chaos execution and status visibility.
- Chaos action framework with execution profiles and configurable action controls.
- Cron-driven chaos execution flow with shared kick orchestration.
- Operator-facing status reporting for queue-table visibility and failure diagnostics.
- PHPUnit coverage for module behavior and diagnostics validation.

### Tooling
- Minimal Harness scaffold for local Claude/Harness project setup.
